# Projeto criado com Create React App



## Scripts disponíveis

Para rodar, acesse o diretório raiz e execute:

### `npm start`

Roda o app em mode de desenvolvimento.
Open [http://localhost:3000](http://localhost:3000) no navegador.

Para testar, execute:

### `npm test`

Roda os testes no terminal.
